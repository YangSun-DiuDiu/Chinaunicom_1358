# 16-设备管理-拓扑图测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 设备拓扑图 |
| 涉及页面 | `views/device/topology/index.vue` |
| API 接口 | 1个（getDeviceTopology） |
| 权限要求 | `device:topology:view` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/device/topology` | `device:topology:query` | 获取设备拓扑树 |

## 2. 前置条件

- 以 admin 账号登录
- 至少存在3台设备且配置了端口绑定关系

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 拓扑树可视化区域 | 自定义递归组件 | 树形显示设备节点和连接线 |
| 设备节点 | 自定义SVG/HTML | 显示设备名称、类型图标、状态颜色 |
| 连接线 | CSS/Canvas绘制 | 层级之间的连接线 |
| 放大按钮 | el-button | 点击放大视图 |
| 缩小按钮 | el-button | 点击缩小视图 |
| 刷新按钮 | el-button | 重新加载拓扑 |
| 空状态提示 | 自定义 | 无拓扑数据时显示引导按钮 |
| 节点详情弹窗 | el-dialog | 点击节点弹出设备详情 |

## 4. 正向测试用例

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| TOPO-001 | 拓扑树加载 | 进入拓扑图页面 | 显示设备拓扑树，节点间有连接线 |
| TOPO-002 | 节点渲染 | 查看各节点 | 每个节点显示设备名称、类型图标 |
| TOPO-003 | 状态颜色 | 查看在线/离线设备节点 | 在线设备绿色、离线设备红色 |
| TOPO-004 | 层级结构 | 查看树状层级 | 父设备在上层，子设备在下层 |
| TOPO-005 | 点击节点查看详情 | 1.点击某个设备节点 | 弹出详情弹窗，显示设备完整信息 |
| TOPO-006 | 详情弹窗内容 | 查看详情弹窗 | 包含设备名称、类型、IP、状态、位置等 |
| TOPO-007 | 缩放-放大 | 1.点击放大按钮 | 视图放大，节点和文字变大 |
| TOPO-008 | 缩放-缩小 | 1.点击缩小按钮 | 视图缩小 |
| TOPO-009 | 刷新 | 1.添加新设备绑定 2.点击刷新 | 拓扑树更新显示新节点 |
| TOPO-010 | 缩放范围限制 | 连续点击放大/缩小 | 缩放有最大/最小值限制 |

## 5. 空状态测试

| 用例ID | 测试名称 | 条件/步骤 | 预期结果 |
|--------|----------|-----------|----------|
| TOPO-E01 | 无拓扑数据 | 1.删除所有端口绑定 2.进入拓扑页 | 显示空状态提示"暂无拓扑数据" |
| TOPO-E02 | 空状态引导 | 在空状态下查看 | 显示"前往设备列表"引导按钮 |
| TOPO-E03 | 空状态引导跳转 | 1.点击引导按钮 | 跳转到设备列表页面 |

## 6. 边界与异常测试

| 用例ID | 测试名称 | 条件/步骤 | 预期结果 |
|--------|----------|-----------|----------|
| TOPO-B01 | 单设备拓扑 | 仅有一台设备无绑定时 | 显示单个根节点 |
| TOPO-B02 | 深层嵌套 | 设备A→B→C→D多层绑定 | 树形层级正确渲染（不溢出） |
| TOPO-B03 | 环形绑定保护 | 尝试绑定A→B且B→A | 后端或前端应防止环形绑定 |
| TOPO-X01 | 网络异常加载 | F12阻止topology接口 | 显示加载失败或错误提示 |
