# 23-访客管理-访客日志测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 访客日志 |
| 涉及页面 | `views/visitor/log/index.vue` |
| API 接口 | 4个（listVisitorLog、getVisitorLog、visitorExit、exportVisitorLog） |
| 权限要求 | `visitor:log:list`、`visitor:log:edit`（记录离开）、`visitor:log:export` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/visitor/log/list` | `visitor:log:list` | 访客记录列表 |
| GET | `/visitor/log/{logId}` | `visitor:log:query` | 记录详情 |
| PUT | `/visitor/log/exit/{logId}` | `visitor:log:edit` | 记录离开时间 |
| GET | `/visitor/log/export` | `visitor:log:export` | 导出访客记录 |

## 2. 前置条件

- 以 admin 账号登录
- 数据库中存在访客记录（含VISITING状态）

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 搜索栏-访客姓名 | el-input | 模糊搜索 |
| 搜索栏-登记类型 | el-select | APPOINTMENT/WALKIN |
| 搜索栏-进入时间 | el-date-picker (daterange) | 日期范围 |
| 导出按钮 | el-button | `v-hasPermi="['visitor:log:export']"` |
| 列显隐控制 | RightToolbar | 复选框下拉选择 |
| 表格列-访客姓名 | el-table-column | 显示 |
| 表格列-访客电话 | el-table-column | 显示 |
| 表格列-访客单位 | el-table-column | 显示 |
| 表格列-被访人 | el-table-column | 显示 |
| 表格列-被访人部门 | el-table-column | 显示 |
| 表格列-登记类型 | el-table-column | APPOINTMENT=预约/WALKIN=现场登记 |
| 表格列-进入时间 | el-table-column | 日期格式化 |
| 表格列-离开时间 | el-table-column | 日期格式化（未离开则显示空） |
| 表格列-通行码 | el-table-column | 显示 |
| 表格列-操作 | el-table-column | 记录离开（仅VISITING状态显示） |

## 4. 正向测试用例

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| VLOG-001 | 访客日志列表加载 | 进入访客日志页面 | 表格加载访客记录 |
