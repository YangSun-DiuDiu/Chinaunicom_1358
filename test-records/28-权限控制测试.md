# 28-权限控制测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 测试范围 | 权限指令、路由权限过滤、数据权限隔离、角色控制、API权限 |
| 测试账号 | admin（超管）、zhaojg（生产部）、qianxm（财务部） |

## 2. 权限体系说明

项目使用 RBAC（角色-菜单-权限）模型：
- 权限字符串格式：`{module}:{entity}:{action}`（如 `system:user:list`）
- 指令控制：`v-hasPermi`、`v-hasRole`
- 路由控制：`route.meta.permissions` 过滤
- API控制：`@PreAuthorize("@ss.hasPermi('...')")`

## 3. v-hasPermi 指令测试

| 用例ID | 测试名称 | 账号 | 页面 | 预期结果 |
|--------|----------|------|------|----------|
| PERM-I01 | 无add权限-按钮隐藏 | 无`system:user:add`账号 | 用户管理 | 新增按钮不渲染（DOM移除） |
| PERM-I02 | 无edit权限-按钮隐藏 | 无`system:user:edit`账号 | 用户管理 | 编辑按钮不渲染 |
| PERM-I03 | 无remove权限-按钮隐藏 | 无`system:user:remove`账号 | 用户管理 | 删除按钮不渲染 |
| PERM-I04 | 有所有权限-admin | admin账号 | 所有页面 | 所有按钮可见（`*:*:*`通配） |
| PERM-I05 | 权限精确匹配 | 有`system:user:list`无`system:user:add` | 用户管理 | 仅新增按钮隐藏，其他可见 |
| PERM-I06 | 按钮各模块独立 | 有设备权限无访客权限 | 设备/访客页面 | 设备页按钮正常，访客页按钮隐藏 |

## 5. 路由权限过滤测试

| 用例ID | 测试名称 | 账号 | 预期结果 |
|--------|----------|------|----------|
| PERM-RT01 | 无权限路由不注册 | 无`device:list`的账号 | 设备管理菜单不显示，直接输入URL无法访问 |
| PERM-RT02 | 通配符管理员 | admin账号 | 所有路由可见 |
| PERM-RT03 | 受限账号菜单 | zhaojg账号 | 侧边栏仅显示有权限的菜单 |
| PERM-RT04 | 直接访问未授权URL | 受限账号在地址栏输入无权限页面URL | 重定向到401或首页 |
| PERM-RT05 | 隐藏路由可访问 | 有`system:user:edit`账号访问分配角色隐藏页 | 可正常访问 |

## 6. API 层权限测试

| 用例ID | 测试名称 | 方法 | 预期结果 |
|--------|----------|------|----------|
| PERM-A01 | 直接调用受保护API | 无权限账号用Postman调用 DELETE /system/user/1 | HTTP 403 Forbidden |
| PERM-A02 | 直接调用公开API | 无权限调用 GET /captchaImage | HTTP 200 正常返回 |
| PERM-A03 | Token过期后调用 | 使用过期token调用任何API | HTTP 401 Unauthorized |
| PERM-A04 | 无Token调用 | 不带Authorization头调用API | HTTP 401 或 403 |
| PERM-A05 | 伪造Token调用 | 使用随机字符串作为token调用API | HTTP 401 |

## 7. 数据权限隔离测试

数据权限通过 `@DataScope` 注解实现，按角色 `dataScope` 字段控制。

| 用例ID | 测试名称 | 账号（dataScope） | 页面 | 预期结果 |
|--------|----------|-------------------|------|----------|
| PERM-D01 | 全部数据权限 | admin（dataScope=1） | 用户管理 | 看到所有部门用户 |
| PERM-D02 | 本部门数据 | zhaojg（生产部） | 用户管理 | 仅看到生产部用户 |
| PERM-D03 | 本部门及以下 | dataScope=4的角色 | 用户管理 | 看到本部门及子部门用户 |
| PERM-D04 | 自定义数据 | dataScope=2的角色 | 用户管理 | 仅看到指定部门的用户 |
| PERM-D05 | 仅本人数据 | dataScope=5的角色 | 用户管理 | 仅看到自己的用户记录 |
