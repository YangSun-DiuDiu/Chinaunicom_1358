# 22-访客管理-现场登记测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 访客现场登记 |
| 涉及页面 | `views/visitor/register/index.vue` |
| API 接口 | 1个主接口（registerVisitor） |
| 权限要求 | `visitor:register:add` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| POST | `/visitor/log` | `visitor:log:add` | 现场登记（创建预约+来访记录） |

## 2. 前置条件

- 以 admin 账号登录
- 存在被访人信息

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 登记表单标题 | 文本 | "访客现场登记" |
| 访客姓名 | el-input | 必填 |
| 访客电话 | el-input | 必填 |
| 访客身份证号 | el-input | 可选 |
| 访客单位 | el-input | 可选 |
| 来访事由 | el-input (textarea) | 可选 |
| 被访人姓名 | el-input | 必填 |
| 被访人部门 | el-input | 必填 |
| 被访人电话 | el-input | 必填 |
| 进入时间 | el-date-picker | 自动填充当前时间，可编辑 |
| 备注 | el-input (textarea) | 可选 |
| 提交按钮 | el-button | 提交登记 |
| 重置按钮 | el-button | 清空表单 |
| 今日登记记录表格 | el-table | 显示今日的登记记录 |
| 登记记录列-访客姓名 | el-table-column | 显示 |
| 登记记录列-访客电话 | el-table-column | 显示 |
| 登记记录列-被访人 | el-table-column | 显示 |
| 登记记录列-进入时间 | el-table-column | 日期格式化 |
| 登记记录列-状态 | el-table-column | VISITING/COMPLETED |

## 4. 正向测试用例

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| REG-001 | 页面加载 | 进入现场登记页面 | 表单加载，进入时间自动填充当前时间，今日记录表格加载 |
| REG-002 | 完整信息登记 | 1.填写所有字段 2.点击提交 | 登记成功，弹出成功弹窗显示访客信息 |
| REG-003 | 必填最小登记 | 1.仅填写必填字段（姓名/电话/被访人信息） 2.提交 | 登记成功 |
| REG-004 | 进入时间默认值 | 1.查看进入时间字段 | 自动填充当前日期时间 |
| REG-005 | 今日记录更新 | 1.新登记一条后查看表格 | 今日记录表格新增该条记录 |
| REG-006 | 成功弹窗信息 | 1.登记成功后查看弹窗 | 显示访客姓名、电话、被访人、进入时间、通行码 |
| REG-007 | 重置表单 | 1.填写部分字段 2.点击重置 | 所有字段清空，进入时间恢复当前时间 |
| REG-008 | 登记类型-WALKIN | 1.提交登记后检查数据库 | registerType 字段值为 WALKIN |
| REG-009 | 登记自动生成通行码 | 1.登记成功后检查 | 通行码自动生成 |
