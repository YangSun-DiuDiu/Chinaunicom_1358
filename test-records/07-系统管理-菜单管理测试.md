# 07-系统管理-菜单管理测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 菜单管理 |
| 涉及页面 | `views/system/menu/index.vue` |
| API 接口 | 8个（list、getMenu、treeselect、roleMenuTreeselect、addMenu、updateMenu、updateMenuSort、delMenu） |
| 权限要求 | `system:menu:list`、`system:menu:add`、`system:menu:edit`、`system:menu:remove` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/system/menu/list` | `system:menu:list` | 菜单列表 |
| GET | `/system/menu/{menuId}` | `system:menu:query` | 菜单详情 |
| GET | `/system/menu/treeselect` | 用户级 | 菜单下拉树 |
| GET | `/system/menu/roleMenuTreeselect/{roleId}` | 用户级 | 角色菜单树 |
| POST | `/system/menu` | `system:menu:add` | 新增菜单 |
| PUT | `/system/menu` | `system:menu:edit` | 修改菜单 |
| PUT | `/system/menu/updateSort` | `system:menu:edit` | 保存排序 |
| DELETE | `/system/menu/{menuId}` | `system:menu:remove` | 删除菜单 |

## 2. 前置条件

- 以 admin 账号登录
- 存在完整的菜单树

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 搜索栏-菜单名称 | el-input | 可输入搜索 |
| 搜索栏-状态 | el-select | 显示/隐藏 |
| 新增按钮 | el-button | `v-hasPermi="['system:menu:add']"` |
| 展开/折叠按钮 | el-button | 全部展开/折叠 |
| 树形表格 | el-table (tree) | 树形结构 |
| 表格列-菜单名称 | el-table-column | 显示，含图标 |
| 表格列-图标 | el-table-column | 显示图标名称 |
| 表格列-排序 | el-table-column | 显示 |
| 表格列-权限标识 | el-table-column | 显示perms字段 |
| 表格列-组件路径 | el-table-column | 显示component |
| 表格列-状态 | el-table-column | 显示/隐藏 |
| 表格列-创建时间 | el-table-column | 显示 |
| 表格列-操作 | el-table-column | 新增子菜单/编辑/删除 |

### 新增/编辑菜单弹窗字段

| 字段 | 类型 | 适用菜单类型 | 说明 |
|------|------|-------------|------|
| 上级菜单 | treeselect | M/C/F | 必选，排除自己 |
| 菜单类型 | el-radio-group | M/C/F | M=目录/C=菜单/F=按钮 |
| 菜单图标 | IconSelect | M/C | 图标选择器 |
| 菜单名称 | el-input | M/C/F | 必填 |
| 显示排序 | el-input-number | M/C/F | 必填 |
| 是否外链 | el-radio | M/C | 是/否 |
| 路由地址 | el-input | M/C | 必填 |
| 组件路径 | el-input | C | 菜单类型需要 |
| 权限标识 | el-input | C/F | 权限字符 |
| 路由参数 | el-input | C | 可选 |
| 是否缓存 | el-radio | C | 是/否 |
| 显示状态 | el-radio | M/C | 显示/隐藏 |
