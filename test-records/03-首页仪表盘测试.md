# 03-首页仪表盘测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 首页仪表盘 |
| 涉及页面 | `views/index.vue`（自定义双模仪表盘）、`views/index_v1.vue`（原始仪表盘） |
| API 接口 | 2个（getDeviceDashboard、getVisitorDashboard） |
| 权限要求 | 需登录即可访问（路径 `/index`） |

### API 清单

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/dashboard/device` | 获取设备驾驶舱数据 |
| GET | `/dashboard/visitor` | 获取访客驾驶舱数据 |

## 2. 前置条件

- 已登录系统（任意账号）
- 数据库中有设备数据（含在线/离线/不同类型）
- 数据库中有访客预约数据（含不同状态、今日数据）

## 3. 页面元素检查

### 3.1 设备驾驶舱模式

| 元素 | 类型 | 检查项 |
|------|------|------|
| 模式切换按钮 | el-radio-group | 设备/访客模式切换 |
| 总设备数卡片 | 统计卡片 | count-to 动画、数值正确 |
| 在线设备数卡片 | 统计卡片 | 绿色高亮、数值正确 |
| 离线设备数卡片 | 统计卡片 | 红色高亮、数值正确 |
| 在线率卡片 | 统计卡片 | 百分比显示正确 |
| 设备类型分布 | ECharts 饼图 | 各类型数量与比例 |
| 最新告警表格 | el-table | 离线设备列表、分页 |
| 最新心跳表格 | el-table | 心跳检测记录、分页 |
| "查看更多"链接 | router-link | 跳转到对应管理页面 |

### 3.2 访客驾驶舱模式

| 元素 | 类型 | 检查项 |
|------|------|------|
| 今日访客统计卡片 | 统计卡片 | 数值正确（今日创建的预约数） |
| 待审批卡片 | 统计卡片 | 数值正确（状态=PENDING的数量） |
| 总访客卡片 | 统计卡片 | 数值正确 |
| 今日进入卡片 | 统计卡片 | 数值正确（今日 REGISTER 类型数） |
| 预约状态分布 | ECharts 柱状图 | 各状态数量对应 |
| 最新来访表格 | el-table | 显示最新访客记录 |
| 待审批列表 | el-table | 显示待审批的预约 |
| "查看更多"链接 | router-link | 跳转到对应管理页面 |

| DASH-E05 | 并发请求 | 快速连续切换设备/访客模式 | 不出现数据错乱或闪烁 |
