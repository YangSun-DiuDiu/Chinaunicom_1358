# 13-监控管理-操作日志测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 操作日志 |
| 涉及页面 | `views/monitor/operlog/index.vue` |
| API 接口 | 4个（list、delOperlog、cleanOperlog、export） |
| 权限要求 | `monitor:operlog:list`、`monitor:operlog:remove`、`monitor:operlog:export` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/monitor/operlog/list` | `monitor:operlog:list` | 操作日志列表 |
| POST | `/monitor/operlog/export` | `monitor:operlog:export` | 导出操作日志 |
| DELETE | `/monitor/operlog/{operIds}` | `monitor:operlog:remove` | 删除操作日志 |
| DELETE | `/monitor/operlog/clean` | `monitor:operlog:remove` | 清空所有操作日志 |

## 2. 前置条件

- 以 admin 账号登录
- 系统中已有操作记录（至少50条）

## 4. 正向测试用例

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| OPER-001 | 操作日志列表加载 | 进入操作日志页面 | 表格加载日志列表，分页正常 |
| OPER-002 | 查看详情 | 1.点击某条日志的"详情"按钮 | 弹出详情弹窗，显示操作URL、请求参数、返回结果等 |
| OPER-003 | 详情-请求参数 | 查看日志详情弹窗 | 请求参数以JSON格式显示 |
| OPER-004 | 详情-返回结果 | 查看日志详情弹窗 | 返回结果显示操作结果信息 |
| OPER-005 | 详情-错误信息 | 查找失败的操作日志查看详情 | 显示错误信息 |
| OPER-006 | 删除单条日志 | 1.选中一条 2.点击删除 3.确认 | 删除成功 |
| OPER-007 | 批量删除日志 | 1.勾选多条 2.点击删除 3.确认 | 批量删除成功 |
| OPER-008 | 清空日志 | 1.点击清空 2.确认（可能需二次确认） | 所有操作日志被清空 |
| OPER-009 | 导出日志 | 点击导出按钮 | 下载操作日志Excel |
| OPER-010 | 按条件导出 | 设置筛选条件后导出 | 仅导出筛选后的日志 |
| OPER-011 | 日志自动记录 | 1.执行任意操作（如新增用户） 2.查看操作日志 | 新操作被记录，包含操作人、时间、IP等 |
