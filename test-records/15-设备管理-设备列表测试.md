# 15-设备管理-设备列表测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 设备列表（设备管理核心页面） |
| 涉及页面 | `views/device/list/index.vue` |
| API 接口 | 8个（listDevice、getDevice、addDevice、updateDevice、delDevice、exportDevice、repairDevice、listDevicePort） |
| 权限要求 | `device:list`、`device:add`、`device:edit`、`device:remove`、`device:export`、`device:repair` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/device/list` | `device:list` | 设备列表（分页） |
| GET | `/device/{deviceId}` | `device:query` | 设备详情 |
| POST | `/device` | `device:add` | 新增设备（@Validated） |
| PUT | `/device` | `device:edit` | 修改设备（@Validated） |
| DELETE | `/device/{deviceIds}` | `device:remove` | 删除设备 |
| POST | `/device/export` | `device:export` | 导出设备Excel |
| POST | `/device/repair/{deviceId}` | `device:repair` | 报修发送短信 |
| GET | `/device/port/list` | `device:port:list` | 设备端口列表 |

## 2. 前置条件

- 以 admin 账号登录
- 数据库中存在多台设备（不同类型/不同状态）
