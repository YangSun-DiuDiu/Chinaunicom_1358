# 24-短信管理-短信配置测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 短信配置 |
| 涉及页面 | `views/sms/config/index.vue` |
| API 接口 | 2个（getConfigKey 读取 + updateConfig 保存，使用系统参数配置接口） |
| 权限要求 | `sms:config:edit` |

### 相关API

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/system/config/configKey/{key}` | 需登录 | 读取单个配置值 |
| PUT | `/system/config` | `system:config:edit` | 修改参数配置 |

使用的配置Key：
- `sms.api.url` — 短信API地址
- `sms.enabled` — 短信开关（true/false）
- `sms.template.deviceOffline` — 设备离线模板
- `sms.template.deviceOnline` — 设备上线模板
- `sms.template.repair` — 报修模板
- `sms.template.visitorApproval` — 访客审批模板

## 2. 前置条件

- 以 admin 账号登录
- 数据库 `sys_config` 表中有短信相关配置

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 短信API地址 | el-input | 可编辑，保存配置 |
| 短信启用开关 | el-switch | 开启/关闭短信功能 |
| 模板-设备离线 | el-input (textarea) | 可编辑模板内容 |
| 模板-设备上线 | el-input (textarea) | 可编辑模板内容 |
| 模板-设备报修 | el-input (textarea) | 可编辑模板内容 |
| 模板-访客审批 | el-input (textarea) | 可编辑模板内容 |
| 变量占位符提示 | 文本 | 显示可用变量如 `{deviceName}`, `{ipAddress}` 等 |
| 保存API配置按钮 | el-button | 保存短信API地址 |
| 保存模板按钮 | el-button | 保存所有模板 |
