# 27-通用功能测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 测试范围 | 所有页面共用的通用组件和功能 |
| 涉及组件 | 分页、搜索栏、右侧工具栏（列显隐、刷新、搜索切换）、全屏、主题、尺寸切换、面包屑、标签页、侧边栏 |

## 2. 分页组件测试

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| COM-P01 | 默认分页 | 进入任意列表页 | 默认第1页，默认每页10条 |
| COM-P02 | 切换页码 | 点击第2页 | 加载第2页数据 |
| COM-P03 | 修改每页条数 | 选择每页20条 | 加载20条数据 |
| COM-P04 | 跳转页码 | 输入页码跳转 | 跳转到指定页 |
| COM-P05 | 总数显示 | 查看分页组件 | 显示"共 N 条" |
| COM-P06 | 上一页/下一页 | 点击上一页/下一页 | 切换到对应页 |
| COM-P07 | 页码禁用状态 | 在第1页点击上一页 | 上一页按钮禁用 |
| COM-P08 | 最后一页 | 跳转到最后一页 | 数据正确，下一页禁用 |

## 3. 搜索栏功能测试

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| COM-S01 | 搜索栏显示/隐藏 | 点击搜索图标 | 搜索栏展开/收起（带动画） |
| COM-S02 | 搜索功能 | 输入条件点击搜索 | 重置页码为1，执行查询 |

| COM-S05 | 回车搜索 | 在搜索框中按回车 | 触发搜索 |
| COM-S06 | 日期范围-仅选开始 | 仅选择开始日期不选结束 | 搜索条件可能无效或仅按开始时间 |
