# 18-设备管理-故障告警测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 故障告警 |
| 涉及页面 | `views/device/fault/index.vue` |
| API 接口 | 2个（list、repair） |
| 权限要求 | `device:fault:list`、`device:fault:repair` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/device/fault/list` | `device:fault:list` | 故障列表（所有OFFLINE记录） |
| GET | `/device/fault/stats` | `device:fault:list` | 故障统计 |
| POST | `/device/fault/repair/{deviceId}` | `device:fault:repair` | 报修发送短信 |

## 2. 前置条件

- 以 admin 账号登录
- 存在离线设备（产生告警记录）
- 存在状态变更日志

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 统计卡片-当前离线设备数 | 统计卡片 | 显示当前OFFLINE的设备数量 |
| 统计卡片-总故障次数 | 统计卡片 | 显示历史离线记录总数 |
| 统计卡片-故障设备列表 | 统计卡片 | 显示涉及故障的设备名称（逗号分隔） |
| 当前离线设备表格 | el-table | 显示当前状态为OFFLINE的设备列表 |
| 告警日志表格 | el-table | 显示所有OFFLINE状态变更记录 |
| 告警表格列-设备名称 | el-table-column | 显示 |
| 告警表格列-旧状态 | el-table-column | 显示 |
| 告警表格列-新状态 | el-table-column | OFFLINE红色显示 |
| 告警表格列-变更时间 | el-table-column | 日期格式化 |
| 告警表格列-短信发送 | el-table-column | Y/N标签显示 |
| 报修按钮 | el-button（每行） | 发送报修短信 |
