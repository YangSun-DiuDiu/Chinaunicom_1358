# 26-个人中心测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 个人中心 |
| 涉及页面 | `views/system/user/profile/index.vue`（含3个子Tab：userInfo、resetPwd、userAvatar） |
| API 接口 | 4个（getUserProfile、updateUserProfile、updateUserPwd、uploadAvatar） |
| 权限要求 | 需登录即可 |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/system/user/profile` | 需登录 | 获取个人信息 |
| PUT | `/system/user/profile` | 需登录 | 修改个人信息 |
| PUT | `/system/user/profile/updatePwd` | 需登录 | 修改密码 |
| POST | `/system/user/profile/avatar` | 需登录 | 上传头像 |

## 2. 前置条件

- 已登录任意账号

## 3. 页面元素检查

### 3.1 基本资料Tab

| 元素 | 类型 | 检查项 |
|------|------|------|
| 用户头像 | 图片 | 显示当前头像或默认头像 |
| 用户名称 | 显示 | 显示登录用户名 |
| 用户昵称 | el-input | 可编辑 |
| 手机号码 | el-input | 可编辑 |
| Email | el-input | 可编辑，格式校验 |
| 性别 | el-radio | 男/女 |
| 保存按钮 | el-button | 保存修改 |

### 3.2 修改密码Tab

| 元素 | 类型 | 检查项 |
|------|------|------|
| 旧密码 | el-input (password) | 必填 |
| 新密码 | el-input (password) | 必填，最短5字符 |
| 确认密码 | el-input (password) | 必填，与新密码一致 |
| 保存按钮 | el-button | 保存新密码 |

### 3.3 头像上传Tab

| 元素 | 类型 | 检查项 |
|------|------|------|
| 当前头像预览 | 图片 | 显示当前头像 |
| 上传按钮/区域 | ImageUpload | 选择并上传头像 |
| 裁剪功能 | cropper | 是否支持裁剪（若有） |

## 5. 边界与异常测试

| 用例ID | 测试名称 | 条件/步骤 | 预期结果 |
|--------|----------|-----------|----------|
| PROFILE-B01 | 昵称超长 | 输入超过30字符昵称 | 前端校验阻止 |
| PROFILE-B02 | 手机号格式 | 输入非11位数字 | 前端校验提示 |
| PROFILE-B03 | Email格式 | 输入非法Email | 前端校验提示 |
| PROFILE-B04 | Email重复 | 修改为其他用户已使用的Email | 提示Email已存在 |
| PROFILE-B05 | 手机号重复 | 修改为其他用户已使用的手机号 | 提示手机号已存在 |
| PROFILE-B06 | 旧密码错误 | 修改密码时输入错误旧密码 | 提示"旧密码错误" |
| PROFILE-B07 | 新密码确认不一致 | 两次输入新密码不同 | 提示"两次密码不一致" |
| PROFILE-B08 | 新密码与旧密码相同 | 新密码与旧密码相同 | 提示"新密码不能与旧密码相同" |
| PROFILE-B09 | 密码长度不足 | 输入少于5字符密码 | 前端校验阻止 |
| PROFILE-B10 | 头像文件格式 | 上传非图片文件 | 前端限制或上传失败 |
| PROFILE-B11 | 头像文件大小 | 上传超大图片（≥10MB） | 提示文件大小超限 |
