# 04-系统管理-用户管理测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 用户管理 |
| 涉及页面 | `views/system/user/index.vue`、`views/system/user/view.vue`、`views/system/user/authRole.vue` |
| API 接口 | 16个（list、getUser、addUser、updateUser、delUser、resetUserPwd、changeUserStatus、export、importData、importTemplate、authRole等） |
| 权限要求 | `system:user:list`、`system:user:add`、`system:user:edit`、`system:user:remove`、`system:user:export`、`system:user:import`、`system:user:resetPwd` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/system/user/list` | `system:user:list` | 用户列表（分页） |
| GET | `/system/user/{userId}` | `system:user:query` | 用户详情 |
| GET | `/system/user/` | `system:user:query` | 获取角色和岗位列表（新增用） |
| GET | `/system/user/deptTree` | `system:user:list` | 部门树 |
| GET | `/system/user/authRole/{userId}` | `system:user:query` | 用户已授权角色 |
| POST | `/system/user` | `system:user:add` | 新增用户 |
| POST | `/system/user/export` | `system:user:export` | 导出Excel |
| POST | `/system/user/importData` | `system:user:import` | 导入Excel |
| POST | `/system/user/importTemplate` | 公开 | 下载导入模板 |
| PUT | `/system/user` | `system:user:edit` | 修改用户 |
| PUT | `/system/user/resetPwd` | `system:user:resetPwd` | 重置密码 |
| PUT | `/system/user/changeStatus` | `system:user:edit` | 修改用户状态 |
| PUT | `/system/user/authRole` | `system:user:edit` | 用户授权角色 |
| DELETE | `/system/user/{userIds}` | `system:user:remove` | 删除用户 |

## 2. 前置条件

- 以 admin 账号登录（拥有全部权限）
- 数据库中存在部门层级结构
- 数据库中存在至少2个角色
