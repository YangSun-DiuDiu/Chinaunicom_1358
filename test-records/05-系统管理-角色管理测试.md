# 05-系统管理-角色管理测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 角色管理 |
| 涉及页面 | `views/system/role/index.vue`、`views/system/role/authUser.vue`、`views/system/role/selectUser.vue` |
| API 接口 | 12个（list、getRole、addRole、updateRole、delRole、dataScope、changeRoleStatus、export、allocatedUserList、unallocatedUserList、authUserCancel/CancelAll、authUserSelectAll） |
| 权限要求 | `system:role:list`、`system:role:add`、`system:role:edit`、`system:role:remove`、`system:role:export` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/system/role/list` | `system:role:list` | 角色列表 |
| GET | `/system/role/{roleId}` | `system:role:query` | 角色详情 |
| GET | `/system/role/deptTree/{roleId}` | `system:role:query` | 数据权限部门树 |
| GET | `/system/role/authUser/allocatedList` | `system:role:list` | 已分配用户列表 |
| GET | `/system/role/authUser/unallocatedList` | `system:role:list` | 未分配用户列表 |
| POST | `/system/role` | `system:role:add` | 新增角色 |
| POST | `/system/role/export` | `system:role:export` | 导出Excel |
| PUT | `/system/role` | `system:role:edit` | 修改角色 |
| PUT | `/system/role/dataScope` | `system:role:edit` | 数据权限设置 |
| PUT | `/system/role/changeStatus` | `system:role:edit` | 状态切换 |
| PUT | `/system/role/authUser/cancel` | `system:role:edit` | 取消单个授权 |
| PUT | `/system/role/authUser/cancelAll` | `system:role:edit` | 批量取消授权 |
| PUT | `/system/role/authUser/selectAll` | `system:role:edit` | 批量授权用户 |
| DELETE | `/system/role/{roleIds}` | `system:role:remove` | 删除角色 |

## 2. 前置条件

- 以 admin 账号登录
- 存在部门层级结构
- 存在多个可分配的用户

## 3. 页面元素检查

### 3.1 主列表页面

| 元素 | 类型 | 检查项 |
|------|------|------|
| 搜索栏-角色名称 | el-input | 可输入搜索 |
| 搜索栏-权限字符 | el-input | 可输入搜索 |
| 搜索栏-状态 | el-select | 正常/停用 |
| 搜索栏-创建时间 | el-date-picker (daterange) | 日期范围 |
| 新增按钮 | el-button | `v-hasPermi="['system:role:add']"` |
| 导出按钮 | el-button | `v-hasPermi="['system:role:export']"` |
| 表格列-角色编号 | el-table-column | 显示 |
| 表格列-角色名称 | el-table-column | 显示 |
| 表格列-权限字符 | el-table-column | 显示 |
| 表格列-显示顺序 | el-table-column | 显示 |
| 表格列-状态 | el-table-column | 开关切换 |
| 表格列-创建时间 | el-table-column | 日期格式化 |
| 表格列-操作 | el-table-column | 编辑/删除/更多（数据权限/分配用户） |

### 3.2 新增/编辑弹窗

| 字段 | 类型 | 校验规则 |
|------|------|----------|
| 角色名称 | el-input | 必填，最长30字符，唯一 |
| 权限字符 | el-input | 必填，最长100字符，唯一 |
| 角色顺序 | el-input-number | 必填 |
| 状态 | el-radio | 正常/停用 |
| 菜单权限 | el-tree (多选) | 分配菜单权限树 |
| 备注 | el-input (textarea) | 可选 |

### 3.3 数据权限弹窗

| 字段 | 类型 | 说明 |
|------|------|------|
| 角色名称 | 显示 | 显示当前角色名称 |
| 权限字符 | 显示 | 显示当前角色权限字符 |
| 数据范围 | el-select | 全部数据/自定义数据/本部门/本部门及以下/仅本人 |
| 指定部门 | el-tree (多选) | 数据范围为"自定义数据"时显示 |

### 3.4 分配用户弹窗

| 元素 | 类型 | 说明 |
|------|------|------|
| 已分配用户标签页 | el-tab-pane | 显示已分配用户列表，可取消 |
| 未分配用户标签页 | el-tab-pane | 显示未分配用户列表，可批量添加 |
