# 25-短信管理-短信日志测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 短信日志 |
| 涉及页面 | `views/sms/log/index.vue` |
| API 接口 | 2个（listSmsLog、getSmsLog） |
| 权限要求 | `sms:log:list` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/sms/log/list` | `sms:log:list` | 短信日志列表 |
| GET | `/sms/log/{smsId}` | `sms:log:query` | 短信日志详情 |

## 2. 前置条件

- 以 admin 账号登录
- 系统已发送过短信（设备告警/报修/访客审批等触发的短信）
- 数据库 `sys_sms_log` 表中有数据

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 搜索栏-收件人 | el-input | 模糊搜索 |
| 搜索栏-手机号码 | el-input | 搜索 |
| 搜索栏-发送结果 | el-select | SUCCESS/FAIL |
| 搜索栏-发送时间 | el-date-picker (daterange) | 日期范围 |
| 搜索按钮 | el-button | 触发查询 |
| 重置按钮 | el-button | 清空筛选条件 |
| 表格列-收件人 | el-table-column | 显示 |
| 表格列-手机号码 | el-table-column | 显示（脱敏显示中间4位） |
| 表格列-短信内容 | el-table-column | 显示（可能截断） |
| 表格列-发送时间 | el-table-column | 日期格式化 |
| 表格列-发送结果 | el-table-column | SUCCESS绿色/FAIL红色 |
| 表格列-业务类型 | el-table-column | DEVICE_OFFLINE/DEVICE_ONLINE/REPAIR/VISITOR_APPROVE |
| 详情按钮 | el-button（每行） | 查看完整短信内容 |

## 4. 正向测试用例

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| SMSL-001 | 短信日志列表加载 | 进入短信日志页面 | 表格加载短信记录 |
| SMSL-002 | 查看详情 | 1.点击某条的详情按钮 | 弹出详情弹窗，显示完整短信内容 |
| SMSL-003 | 详情-业务类型 | 查看详情弹窗 | 显示bizType和关联的bizId |
| SMSL-004 | 设备离线短信 | 查看bizType=DEVICE_OFFLINE的记录 | 内容包含设备名称、IP、时间 |
| SMSL-005 | 设备上线短信 | 查看bizType=DEVICE_ONLINE的记录 | 内容包含设备上线信息 |
| SMSL-006 | 报修短信 | 查看bizType=REPAIR的记录 | 内容包含报修设备信息 |
| SMSL-007 | 访客审批短信 | 查看bizType=VISITOR_APPROVE的记录 | 内容包含访客信息、通行码链接 |

## 5. 数据筛选测试

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| SMSL-F01 | 按收件人搜索 | 输入收件人名称 | 返回匹配记录 |
| SMSL-F02 | 按手机号码搜索 | 输入完整手机号 | 返回匹配记录 |
| SMSL-F03 | 按发送结果-成功 | 选择SUCCESS | 仅显示成功记录 |
| SMSL-F04 | 按发送结果-失败 | 选择FAIL | 仅显示失败记录 |
| SMSL-F05 | 按发送时间范围 | 选择日期范围 | 返回该时间段记录 |
| SMSL-F06 | 组合筛选 | 手机号+结果+时间 | 返回同时满足条件的记录 |

## 6. 边界与异常测试
