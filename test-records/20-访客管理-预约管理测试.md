# 20-访客管理-预约管理测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 访客预约管理 |
| 涉及页面 | `views/visitor/appointment/index.vue` |
| API 接口 | 10个（list、getAppointment、addAppointment、updateAppointment、delAppointment、approveAppointment、cancelAppointment、completeAppointment、pendingAppointment、export） |
| 权限要求 | `visitor:appointment:list`、`visitor:appointment:add`、`visitor:appointment:edit`、`visitor:appointment:remove`、`visitor:appointment:cancel`、`visitor:appointment:complete`、`visitor:appointment:approve`、`visitor:appointment:export` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/visitor/appointment/list` | `visitor:appointment:list` | 预约列表 |
| GET | `/visitor/appointment/{appointmentId}` | `visitor:appointment:query` | 预约详情 |
| GET | `/visitor/appointment/pending` | `visitor:appointment:pending` | 待审批列表 |
| POST | `/visitor/appointment` | `visitor:appointment:add` | 新增预约（自动生成通行码） |
| PUT | `/visitor/appointment` | `visitor:appointment:edit` | 修改预约 |
| DELETE | `/visitor/appointment/{ids}` | `visitor:appointment:remove` | 删除预约 |
| PUT | `/visitor/appointment/approve` | `visitor:appointment:approve` | 审批预约 |
| PUT | `/visitor/appointment/cancel/{id}` | `visitor:appointment:cancel` | 取消预约 |
| PUT | `/visitor/appointment/complete/{id}` | `visitor:appointment:complete` | 完成来访 |
| POST | `/visitor/appointment/export` | `visitor:appointment:export` | 导出预约Excel |

## 2. 前置条件

- 以 admin 账号登录
- 存在被访人信息（部门、电话）

## 5. 状态流转测试

访客预约状态流转: PENDING → APPROVED → VISITING → COMPLETED（或被 CANCELLED/REJECTED 中断）

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| APPT-S01 | PENDING→APPROVED | 1.对待审批预约执行审批通过 | 状态变为APPROVED，自动创建VISITING记录，发送短信 |
| APPT-S02 | PENDING→REJECTED | 1.对预约执行审批拒绝 | 状态变为REJECTED |
| APPT-S03 | PENDING→CANCELLED | 1.访客取消预约 | 状态变为CANCELLED |
| APPT-S04 | APPROVED→CANCELLED | 1.取消已通过的预约 | 状态变为CANCELLED |
| APPT-S05 | VISITING→COMPLETED | 1.点击"完成" | 状态变为COMPLETED，记录离开时间 |
| APPT-S06 | 不可取消已完成的预约 | 查看COMPLETED预约的操作按钮 | 取消按钮不显示 |
| APPT-S07 | 不可取消已取消的预约 | 查看CANCELLED预约的操作按钮 | 取消按钮不显示 |
| APPT-S08 | REJECTED不可操作 | 查看REJECTED预约的操作按钮 | 仅显示删除 |
