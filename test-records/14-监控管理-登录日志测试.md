# 14-监控管理-登录日志测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 登录日志 |
| 涉及页面 | `views/monitor/logininfor/index.vue` |
| API 接口 | 5个（list、delLogininfor、cleanLogininfor、export、unlock） |
| 权限要求 | `monitor:logininfor:list`、`monitor:logininfor:remove`、`monitor:logininfor:export`、`monitor:logininfor:unlock` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/monitor/logininfor/list` | `monitor:logininfor:list` | 登录日志列表 |
| POST | `/monitor/logininfor/export` | `monitor:logininfor:export` | 导出登录日志 |
| DELETE | `/monitor/logininfor/{infoIds}` | `monitor:logininfor:remove` | 删除登录日志 |
| DELETE | `/monitor/logininfor/clean` | `monitor:logininfor:remove` | 清空登录日志 |
| GET | `/monitor/logininfor/unlock/{userName}` | `monitor:logininfor:unlock` | 解锁用户账户 |

## 2. 前置条件

- 以 admin 账号登录
- 系统中有登录记录（成功和失败都有）
