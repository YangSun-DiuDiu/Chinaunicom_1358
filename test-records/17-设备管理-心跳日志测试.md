# 17-设备管理-心跳日志测试

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 模块名称 | 心跳日志 |
| 涉及页面 | `views/device/heartbeat/index.vue` |
| API 接口 | 2个（listHeartbeatLog 查询 + export 导出） |
| 权限要求 | `device:heartbeat:list` |

### API 清单

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/device/heartbeat/list` | `device:heartbeat:list` | 心跳日志列表 |
| POST | `/device/heartbeat/export` | `device:heartbeat:export` | 导出心跳日志 |

## 2. 前置条件

- 以 admin 账号登录
- 已部署心跳检测定时任务并运行了一段时间
- 数据库 `iot_device_heartbeat_log` 表中有数据

## 3. 页面元素检查

| 元素 | 类型 | 检查项 |
|------|------|------|
| 搜索栏-设备名称 | el-input | 模糊搜索 |
| 搜索栏-Ping结果 | el-select | SUCCESS/FAIL |
| 搜索栏-检测时间 | el-date-picker (daterange) | 日期范围 |
| 搜索按钮 | el-button | 触发查询 |
| 重置按钮 | el-button | 清空筛选条件 |
| 导出按钮 | el-button | 导出Excel（无显式权限检查） |
| 表格列-设备名称 | el-table-column | 显示 |
| 表格列-IP地址 | el-table-column | 显示 |
| 表格列-Ping结果 | el-table-column | SUCCESS绿色/FAIL红色 |
| 表格列-Ping延迟(ms) | el-table-column | 显示毫秒数 |
| 表格列-检测时间 | el-table-column | 日期格式化 |
| 分页组件 | Pagination | 分页正常 |

## 4. 正向测试用例

| 用例ID | 测试名称 | 步骤 | 预期结果 |
|--------|----------|------|----------|
| HB-001 | 心跳日志列表加载 | 进入心跳日志页面 | 表格加载心跳记录 |
| HB-002 | 日志自动生成 | 1.等待定时任务执行 2.刷新心跳日志页 | 新增心跳记录，状态随在线/离线变化 |
| HB-003 | Ping成功记录 | 查看在线设备的心跳记录 | pingResult=SUCCESS |
| HB-004 | Ping失败记录 | 查看离线设备的心跳记录 | pingResult=FAIL |
| HB-005 | Ping延迟显示 | 查看SUCCESS记录 | pingLatency显示具体毫秒值 |
| HB-006 | 导出心跳日志 | 1.点击导出按钮 | 下载Excel文件 |
| HB-007 | 按条件导出 | 1.筛选条件后导出 | 仅导出筛选结果 |

## 5. 数据筛选测试

| 用例ID | 测试名称 | 步骤 | 预期结果 |
