### 测试策略

1. **先正向后异常**：先验证正常流程，再测试边界和异常场景
2. **先独立后组合**：先测试单个功能，再测试功能组合
3. **分层测试**：页面交互 → API 数据 → 数据库一致性
4. **权限覆盖**：使用不同权限账号测试权限控制有效性
