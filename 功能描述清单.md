# 智慧园区管理系统 — 功能描述清单

> 版本: v1.0 | 日期: 2026-06-26 | 基于 RuoYi-Vue 3.9.2 深度定制

---

## 一、系统基础

| 功能 | 说明 |
|------|------|
| 用户管理 | 用户 CRUD、角色分配、部门归属、密码重置、导入导出 |
| 部门管理 | 树形组织架构 CRUD、多租户数据隔离基础 |
| 角色管理 | 角色 CRUD、菜单权限分配、数据权限范围（全部/自定义/本部门/及子部门/本人） |
| 菜单管理 | 树形菜单 CRUD、目录/菜单/按钮三级、权限标识绑定 |
| 岗位管理 | 岗位 CRUD、用户岗位关联 |
| 字典管理 | 字典类型 + 字典数据 CRUD、Redis 缓存、前端自动翻译 |
| 参数配置 | 系统参数键值对 CRUD、Redis 缓存 |
| 通知公告 | 公告 CRUD + 已读标记 + 批量标记已读 |
| 日志管理 | 操作日志 + 登录日志，支持查询/导出/清空 |

## 二、多租户数据隔离

| 功能 | 说明 |
|------|------|
| 组织数据隔离 | Dept = 租户，用户自动绑定所属组织 |
| 自动过滤 | MyBatis 拦截器自动为业务表 SELECT 追加 `dept_id` 过滤 |
| 自动填充 | AOP 切面为新增/修改数据自动填充 `dept_id` |
| 组织切换 | 管理员可切换组织视角查看不同组织数据 |
| 白名单机制 | 系统表（用户/角色/菜单/日志等）不参与过滤 |

## 三、设备管理

| 功能 | 说明 |
|------|------|
| 设备列表 | IoT 设备 CRUD、在线/离线状态统计、类型统计、拓扑树 |
| 心跳监测 | 设备 Ping 检测 + 心跳日志记录（Quartz 定时任务） |
| 故障预警 | 设备故障告警记录 |
| SNMP 管理 | SNMPv2c 系统信息/端口/OID 查询（Java 原生回退） |
| 维修工单 | 创建/转派/接单/拒绝/完成，自动生成维修单号和 Token |
| 维修反馈 | 公开 H5 页面，输入 Token 查看工单 + 上传维修照片 |
| 备件管理 | 配件 CRUD + 使用记录查询 |
| 离线告警 | 心跳检测到离线自动创建工单 + 发送短信 |

## 四、智能化管理

| 功能 | 说明 |
|------|------|
| 人员通行设备 | 闸机/门禁设备 CRUD + 一键 Ping 检测 |
| 车辆通行设备 | 车辆道闸 CRUD + 一键 Ping 检测 |
| 通行权限管理 | 按人员/部门授权通行设备及时段 |
| 视频监控设备 | 摄像头 CRUD + RTSP 流 + WebSocket 实时推送 |
| 视频预览 | WebSocket 直播 + 截图快照 |
| 地图模式 | 底图管理 + 设备位置标注 |
| 一键检测 | 批量 Ping 所有智能设备 + 离线自动报修 |

## 五、访客管理

| 功能 | 说明 |
|------|------|
| 访客预约 | 预约 CRUD + 审批流程（通过/拒绝/取消/完成） |
| 访客审批 | 独立审批页面，PENDING 状态筛选 |
| 现场登记 | 现场快速登记生成通行凭证 |
| H5 自助 | 访客自主扫码登记（无需登录），生成 8 位通行码 |
| 通行验证 | 门禁扫码验证通行 |
| 来访记录 | 进出记录查询 + 导出 |

## 六、会议管理

| 功能 | 说明 |
|------|------|
| 会议室管理 | 会议室 CRUD + 平板地址一键复制 |
| 会议预约 | 预约 CRUD + 时间轴可视化 + 冲突检测 |
| 会议审批 | 独立审批页面 + 通过/拒绝 |
| 会议管理 | 今日预约列表 + 日期切换 + 会议室筛选 |
| 平板展示 | 公开 H5 页面，30 秒自动刷新 + 倒计时 + 状态显示 |

## 七、考勤管理

| 功能 | 说明 |
|------|------|
| 数据对接 | 钉钉/企业微信 Open API 配置 + 连接测试 |
| 考勤数据 | 打卡记录查询 + 导出 |
| 回调接收 | Webhook 回调接口（钉钉/企微）+ WebSocket 实时推送 |
| 组织同步 | 钉钉/企微部门 + 用户一键同步到本地 |

## 八、短信管理

| 功能 | 说明 |
|------|------|
| 短信配置 | 阿里云 AccessKey + 签名 + 模板 CODE 配置 |
| 短信日志 | 发送记录查询，按收件人/类型/时间筛选 |
| 业务场景 | 设备离线告警、维修工单通知、访客审批通知 |
| 测试发送 | 配置页一键测试短信通道 |

## 九、仪表盘

| 功能 | 说明 |
|------|------|
| 设备仪表盘 | 在线率、设备类型分布、告警统计（ECharts） |
| 访客仪表盘 | 预约趋势、来访统计 |
