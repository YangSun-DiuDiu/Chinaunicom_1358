# 基于若依分离版定制轻量化管理系统
## 一、项目概述
本项目基于**最新版若依 RuoYi-Vue 分离版框架**深度定制重构，保留模块化开发架构、Maven 构建、SpringBoot 后端、Vue+ElementUI 前端技术栈，新增 H5 移动端适配、微信服务号集成能力、无状态极简登录、设备拓扑监控、设备心跳检测短信告警、完整访客管理体系。

整体**极致精简原生若依冗余功能**，聚焦【组织权限基础模块 + 智能设备管理模块 + 访客管理模块 + 双驾驶舱大屏】四大核心业务，代码轻量化、架构标准化、适配企业内网设备安防管理场景，支持全自动部署、全自动化测试、全文档输出。

## 二、核心开发技能要求（AI 专属开发 Skill）
为保证代码质量、架构统一、功能落地，AI 开发需具备以下专项开发技能，全程全自动执行、无交互询问、无开发中断：
1. **若依框架深度裁剪重构技能**：支持彻底删除若依原生字典管理、代码生成器、在线用户、监控等全部冗余模块，保留核心用户 / 部门 / 角色基础架构。
2. **SpringBoot 无状态 Token 登录改造技能**：精简登录链路、精简动态路由与静态路由结合的方式、Token 过期自动刷新、无状态会话管理。
4. **网络设备运维开发技能**：Java 实现定时任务、 Ping 连通性检测、SNMP 协议设备读写、设备层级拓扑图生成、设备心跳定时巡检、通过双向绑定实现状态实时刷新。
5. **消息告警开发技能**：集成模拟短信 API、设备离线 / 上线短信通知、设备状态变更日志全链路记录、运维恢复台账记录。
6. **Vue+ElementUI 开发技能**：静态路由全局配置、侧边栏定制、双驾驶舱大屏适配、H5 自适应页面开发、微信服务号页面适配。
7. **模块化 Maven 工程优化技能**：精简依赖、剔除无用 Jar 包、统一版本号、工程结构标准化。
8. **数据库设计与迁移技能**：根据实际需求删除无用的数据库表、修改原有数据库表。
9. **自动化测试与文档生成技能**：自动编写单元测试、接口测试、业务测试用例，自动输出接口文档、数据库文档、测试报告。

## 三、整体技术架构
### 3.1 后端架构
- 基础框架：最新版 RuoYi-Vue 分离版

## 四、核心功能定制需求
### 4.1 登录架构全局重构（核心改造）
1. 保留 JWT Token 无状态登录模式，全程无 Session、无服务端会话存储。
2. **彻底精简动态路由逻辑**，前端全部采用**动态+静态路由配置文件**，根据权限对页面进行渲染啊。
3. 精简登录校验逻辑，去除冗余权限校验、菜单加载逻辑，提升登录速度。
4. 保留用户系统细粒度权限控制、按钮权限、数据权限功能。

### 4.2 系统菜单极简裁剪
彻底删除若依原生所有多余模块，系统仅保留 5几大核心菜单：
1. 人员管理
2. 部门管理
3. 角色管理
4. 设备管理
5. 访客管理
6. 日志管理
7. 系统管理

### 4.3 首页驾驶舱（双模式可配置）
1. 系统登录默认首页为数据驾驶舱大屏。
2. 通过**yml 配置文件参数**动态切换首页模式：
    - 模式一：设备管理驾驶舱（设备在线率、设备类型统计、离线告警、拓扑概览、巡检记录）
    - 模式二：访客管理驾驶舱（今日访客、预约访客、历史来访、进出记录统计）
3. 驾驶舱具备行业主流可视化功能：统计图、数据卡片、告警提示、最新记录列表、趋势图表。

### 4.4 基础组织模块（人员 / 部门 / 角色）
1. 沿用若依原生人员、部门、角色底层架构与 UI 样式。
2. 角色权限绑定、菜单权限分配、数据权限过滤。
3. 所有账号登录后权限限制。

### 4.6 设备管理模块（核心业务）
#### 4.6.1 基础功能
- 设备新增、编辑、删除、批量删除、查询、导出、筛选、
- 设备分类管理：网络设备、监控设备、其他设备 三大类型
- 设备基础信息：名称、IP、型号、位置、负责人、端口信息、状态、租户 ID
- 设备离线告警：在设备管理界面，设备由在线状态改为离线状态或者由离线状态恢复到在线状态，均会发送短信。离线后显示报修按钮，用于向维修人员发送维护的短信，恢复在线后，保修按钮隐藏。
- 设备状态：设备状态变更均形成记录。

#### 4.6.2 高级拓扑功能
- 支持根据选择设备选择上下级连接端口配置
- 支持设备上下级端口绑定、链路关联
- 自动根据设备端口连接关系**生成可视化网络拓扑图**
- 拓扑图支持缩放、查看详情、状态高亮

#### 4.6.3 设备心跳检测与告警
1. 后台定时任务轮训 Ping 检测设备在线状态。
2. 设备离线瞬间触发**模拟短信 API**，向设备绑定负责人发送离线通知。
3. 设备重新上线自动记录恢复日志，留存**故障 - 恢复完整台账**。
4. 所有设备状态变更（离线 / 在线）永久留痕，支持查询、统计、溯源。
5. 7权限管理-设备管理作为人员权限进行管理，新增设备管理员权限。

#### 4.6.4 SNMP 设备管理
- 支持 SNMP 协议读取设备信息、设备状态、端口状态
- 支持主流网络设备、安防监控设备兼容适配
- snmp指令可配置。

    - 适配 Ping 连通性检测、SNMP 信息读取、在线状态展示
2. 离线设备：海康威视摄像头 | IP：116.148.212.151
    - 适配离线检测、自动短信告警、离线日志、上线恢复台账

### 4.7 访客管理模块
1. 访客预约管理：新增预约、审核、预约列表、预约作废，在已审批的状态下新增短信通知功能，用于向来访人发送审批通过短信。
2. 现场访客登记：临时来访登记、人脸 / 身份信息录入（预留字段）
3. 来访记录：自动记录来访时间、离开时间、访问事由、访问对象
4. 记录查询、筛选、导出、统计
5. **预留第三方通行设备对接接口**（门禁、闸机、访客终端），可后续拓展硬件联动
6. 访客管理-增加访客审批页面，只有具备访客审批权限的人员才会显示内容。
7. 权限管理-访客审批作为人员权限进行管理，新增访客管理员权限。


### 4.8 移动端适配
1. 基于 ElementUI 开发 H5 自适应页面
2. 全功能适配手机端浏览、操作
3. 页面兼容微信内置浏览器，支持**嵌入微信服务号直接访问**
