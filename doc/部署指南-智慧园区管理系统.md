### Step 1: 准备部署包

```
zhihuiyuanqu20260625/
├── ruoyi-admin.jar        # SpringBoot 后端 JAR (89MB)
```

### Step 6: 配置 Nginx

```bash
# 复制配置文件
cp /root/zhihuiyuanqu20260625/nginx.conf /etc/nginx/conf.d/zhihuiyuanqu.conf

# 测试配置
nginx -t

# 重载 Nginx
nginx -s reload
# 或: systemctl reload nginx
```

**Nginx 配置内容** (`/etc/nginx/conf.d/zhihuiyuanqu.conf`):

```nginx
server {
    listen 80;
    server_name _;

    # 前端静态文件
    root /usr/share/nginx/zhihuiyuanqu/dist;
    index index.html;

    # 前端路由 (Vue Router history 模式)
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 代理到后端 SpringBoot
    location /prod-api/ {
        proxy_pass http://127.0.0.1:8090/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # WebSocket 代理 (考勤推送 + 视频流)
    location /ws/ {
        proxy_pass http://127.0.0.1:8090/ws/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### Step 7: 防火墙配置

```bash
# 通用端口检查
netstat -tlnp | grep -E "80|8090|3306|6379"

# firewalld (CentOS/RHEL)
firewall-cmd --add-port=80/tcp --permanent
firewall-cmd --add-port=8090/tcp --permanent
firewall-cmd --reload

# ufw (Ubuntu)
ufw allow 80/tcp
ufw allow 8090/tcp
```

> ⚠️ **云安全组**: 阿里云/腾讯云服务器还需在控制台安全组中放行 80 和 8090 端口

### Step 8: 启动后端服务

```bash
cd /root/zhihuiyuanqu20260625
chmod +x start-server.sh stop-server.sh

# 启动
bash start-server.sh

# 等待约15秒后检查
sleep 15
curl -s -o /dev/null -w "HTTP %{http_code}\n" http://localhost:8090/
# 预期输出: HTTP 200
```

**启动脚本** (`start-server.sh`):

```bash
#!/bin/bash
cd "$(dirname "$0")"
nohup java -jar ruoyi-admin.jar \
  --server.port=8090 \
  --spring.devtools.restart.enabled=false \
  > app.log 2>&1 &
echo "Backend PID: $!"
```

**停止脚本** (`stop-server.sh`):

```bash
#!/bin/bash
PID=$(ps aux | grep "ruoyi-admin.jar" | grep -v grep | awk '{print $2}')
if [ -n "$PID" ]; then
  kill $PID
  echo "Backend stopped (PID: $PID)"
else
  echo "No running backend found"
fi
```
